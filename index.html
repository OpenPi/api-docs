/authorize

/authorize post

post

Get an authentication token.

post /authorize

Get an authentication token.

  • Request
  • Response

Body

Type: application/json

Schema:

{
  "type": "object",
  "$schema": "http://json-schema.org/draft-03/schema",
  "id": "http://jsonschema.net",
  "required": true,
  "properties": {
    "username": {
      "type": "string",
      "required": true
    },
    "password": {
      "type": "string",
      "required": true
    }
  }
}

Example:

{
  "username": "plashboard",
  "password": "1234"
}

HTTP status code 200

Body

Type: application/json

Schema:

{
  "type": "object",
  "$schema": "http://json-schema.org/draft-03/schema",
  "id": "http://jsonschema.net",
  "required": true,
  "properties": {
    "token": {
      "type": "string",
      "description": "A JWT token"
    }
  }
}

Example:

{
  "token": "ASDFGHJKL"
}

HTTP status code 401

Body

Type: application/json

Schema:

{
  "type": "object",
  "$schema": "http://json-schema.org/draft-03/schema",
  "id": "http://jsonschema.net",
  "required": true,
  "properties": {
    "code": {
      "type": "integer",
      "description": "Error code",
      "required": true
    },
    "message": {
      "type": "string",
      "description": "User friendly error message.",
      "required": true
    }
  }
}

Example:

{
  "code": -1,
  "message": "Username and password invalid"
}

/sensors/{sensorId} get put

Interactions with a specific sensor.

get

Get the sensor with sensorId = {sensorId}.

put

Set the sensor settings.

get /sensors/{sensorId}

Get the sensor with sensorId = {sensorId}.

  • Request
  • Response

URI Parameters

  • sensorId: required (string)

Headers

  • Authorization: required (string)

    The JWT token provided by the /authorize method.

HTTP status code 200

Body

Type: application/json

Schema:

{
  "type": "object",
  "$schema": "http://json-schema.org/draft-03/schema",
  "id": "http://jsonschema.net",
  "required": true,
  "properties": {
    "sensorId": {
      "type": "string",
      "description": "Sensor ID",
      "required": true,
      "minLength": 36,
      "maxLength": 36
    },
    "value": {
      "type": "string",
      "description": "Sensor value",
      "required": true
    }
  }
}

Example:

{
  "sensorId": "temperature",
  "value": 40,
}
